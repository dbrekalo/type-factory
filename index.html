Type factory

Type factory is a convenient generator of javascript prototype based constructor functions with simple and easy to use syntax. Types can be easily extended and used as basis for application modules of any kind. Clocks under 1KB.

Type factory is just a small helper function through which a constructor function can be defined without the usual verbosity. Each generated type can be easily extended with no need for manual setup of prototype chain. Can be used with CommonJS and AMD modules or browser globals.
