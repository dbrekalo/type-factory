Type factory

Type factory is a convenient generator of javascript prototype based constructor functions with simple and easy to use syntax. Types can be easily extended and used as basis for modules of any purpose. It weighs less than 1KB.

Type factory is a small helper function by which a constructor function can be defined without the usual verbosity. Each generated type can be easily extended with no need for manual setup of prototype chain. Can be used with CommonJS and AMD modules or browser globals.
